Review Scrapper

"Review Scrapper" is a web-based project that utilizes Python, Flask, and DBMS (Database Management System) technology. It extracts reviews from Flipkart, automating the collection process for user convenience. The project enables sentiment analysis and opinion mining, providing valuable insights for businesses and consumers. By leveraging web scraping techniques and these technologies, it showcases efficient data extraction and storage capabilities.

Booking System

The "Booking System Interface" is a user-friendly project developed in C++ with a focus on Object-Oriented Programming (OOP). It allows users to book flights, hotels, and rental vehicles seamlessly. The project's utilization of C++ and OOP principles enhances code organization and reusability. With its intuitive interface, users can easily browse, select, and book their travel options, making travel booking convenient and hassle-free.
